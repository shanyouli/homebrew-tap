# Shanyouli Tap

## How do I install these formulae?

`brew install shanyouli/tap/<formula>`

Or `brew tap shanyouli/tap` and then `brew install <formula>`.

Or, in a [`brew bundle`](https://github.com/Homebrew/homebrew-bundle) `Brewfile`:

```ruby
tap "shanyouli/tap"
brew "<formula>"
```
